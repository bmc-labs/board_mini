# Getting Started with the mini::base
the **mini::base** is suitable for a variety of applications and can be used
within different environments   
- Command line based  
- Arduino  
	* bare metal  
	* hid bootloader  
- Platformio  
	* bare metal  
	* bootloader  
- Zephyr 
	* bare metal  
	* mcuboot
